### 💻 **System Requirements**
| Platform | Minimum Requirements |
|----------|---------------------|
| **🍎 macOS** | macOS 10.15 (Catalina) or later, Intel or Apple Silicon (M1/M2/M3) |
| **🪟 Windows** | Windows 10 or later, 64-bit processor |
| **🐧 Linux** | Ubuntu 18.04+, Debian 10+, or equivalent distribution |
| **💾 Storage** | 300 MB free disk space |
| **🌐 Network** | Internet connection for audio streaming and updates |

### 📋 **Installation Guide**

#### **macOS Installation**
1. Download `PlayOuran-1.5.0-Universal.dmg`
2. Open the DMG file and drag **PlayOuran** to Applications
3. Launch from Applications or press `Cmd+Space` and type "PlayOuran"
4. If prompted by security, right-click → "Open" to allow first launch

#### **Windows Installation**
1. Download `PlayOuran-Setup-1.5.0.exe`
2. Run the installer as Administrator
3. Follow the setup wizard
4. Launch from Start Menu or Desktop shortcut

#### **Linux Installation**
```bash
# Download and install .deb package (Ubuntu/Debian)
wget https://github.com/yanis/PlayOuran/releases/latest/download/PlayOuran-1.5.0.deb
sudo dpkg -i PlayOuran-1.5.0.deb

# Or use AppImage (Universal Linux)
wget https://github.com/yanis/PlayOuran/releases/latest/download/PlayOuran-1.5.0.AppImage
chmod +x PlayOuran-1.5.0.AppImage
./PlayOuran-1.5.0.AppImage
```

## 🏗️ **Development Setup & Contributing**

### 🚀 **Quick Start for Developers**

```bash
# 1. Clone the repository
git clone https://github.com/yanis/PlayOuran.git
cd PlayOuran

# 2. Install dependencies
npm install

# 3. Start development server with hot reload
npm start

# 4. Development commands
npm run build              # Build for production
npm run build:mac          # Build for macOS only
npm run build:universal    # Create universal macOS build (Intel + Apple Silicon)
npm run test               # Run unit tests
npm run lint               # Check code quality
```
