![PlayOuran App Screenshot](assets/images/app-preview.png)

## ✨ Features

### 📖 Quran Reading Experience
- **Complete Quran text** with beautiful Arabic typography
- **Multiple reciters** with high-quality audio recitations
- **Surah navigation** with instant chapter selection
- **Responsive design** that adapts to any window size
- **Bookmark system** to save your reading progress

### 🕌 Islamic Utilities
- **Prayer times** with accurate calculations for your location
- **Daily Hadith & Duas** with authentic sources
- **Qibla direction** compass feature
- **Islamic calendar** integration
- **Beautiful Islamic themes** with dark mode support

### 🌟 Modern App Experience
- **Native macOS design** with seamless integration
- **Real-time weather** updates for your location
- **Notification system** for prayer times and reminders
- **Offline support** for uninterrupted spiritual practice
- **Multi-language support** (Arabic, English, French)

## 🚀 Download & Installation

### Direct Download
Download the latest version from our [Releases page](https://github.com/PlayOuran/PlayOuran/releases/latest).

### System Requirements
- **macOS 10.15** (Catalina) or later
- **64-bit Intel** or **Apple Silicon** (M1/M2) processor
- **200 MB** free disk space
- **Internet connection** for audio streaming and weather updates

### Installation Steps
1. Download `PlayOuran-1.5.0-Universal.dmg` from the releases page
2. Open the downloaded DMG file
3. Drag **PlayOuran** to your Applications folder
4. Launch PlayOuran from Applications or Spotlight

> **Note**: On first launch, you may need to right-click and select "Open" due to macOS security settings.

## 🛠️ Built With

- **[Electron](https://electronjs.org/)** - Cross-platform desktop framework
- **[Node.js](https://nodejs.org/)** - JavaScript runtime
- **HTML5 & CSS3** - Modern web technologies
- **JavaScript ES6+** - Application logic and API integrations

### APIs & Services
- **[Quran.com API](https://quran.com)** - Quran text and translations
- **[Alquran.cloud](https://alquran.cloud)** - Audio recitations
- **[Aladhan API](https://aladhan.com)** - Prayer times calculation
- **[OpenWeatherMap](https://openweathermap.org)** - Weather data
- **[HadeethEnc.com](https://hadeethenc.com)** - Authentic Hadith collection

## 🏗️ Development Setup

```bash
# Clone the repository
git clone https://github.com/PlayOuran/PlayOuran.git
cd PlayOuran

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Create macOS universal build
npm run build:universal
```
